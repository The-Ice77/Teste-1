O que é CSS?

CSS significa Cascading Style Sheets ou, em português, Folhas de Estilo em Cascata...

Para que serve e como funciona?

O CSS permite criar páginas web visualmente atraentes...

Quais são as vantagens do CSS?

  • Reutilização de estilos.
  • Responsividade.
  • Eficácia.

Como o CSS é estruturado?

O CSS é composto por regras de estilo com seletor e declarações...
